# Orchard Adopters

This file lists individuals, teams, and organizations that use Orchard, and how
they use it. It exists so that adoption is recorded honestly and with consent.

> A GitHub star is appreciated, but it signals interest - not usage or
> endorsement. **This file is the source of truth for who actually runs
> Orchard.** Everyone listed here has opted in themselves.

## Add yourself

We'd love to have you on the list. Either:

- Open a pull request adding a row to the table below, or
- [Open an issue](https://github.com/andrew-waters/orchard/issues/new) with your
  details and we'll add you.

Only include what you're comfortable making public. Company entries should come
from someone authorized to represent that the organization uses Orchard - please
don't add an employer on their behalf without that.
